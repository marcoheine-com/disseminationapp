Mendeley Publications

This page shows the usage of the Mendeley API and the Mendeley JavaScript SDK. The API is used to get all available Information and all Publications from a Users Mendeley Profile.

This function loads all the data of a mendeley profile:

functiongetProfile() {
MendeleySDK.API.profiles.retrieveByEmail(

'myEmail@email.com'

)

}

This function loads all publications of a users mendeley library:

functiongetDocuments() {
MendeleySDK.API.documents

.list({

authored: true

})

}

In the following the loaded data is displayed. The Publications can be searched and several statistics are provided.

Here needs to be an addition concerning localstorage and storing the data/documents on the server

Afterwards, for every new publication an automated tweet, Browser Push Notification and RSS Feed is served.
