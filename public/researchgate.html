ResearchGate Publications

This page shows the usage of the ResearchGate Plugins.

Unfortunately ResearchGate does not offer an API at the moment. In 2011 they anounced to be working on it and in 2013 a timely release of an API was proclaimed, but since then there were no other news about a ResearchGate API.

They do offer at least plugins, which can be implemented on the website. These plugins fetch different sorts of data. Most interestingly they load all the publications of a department or institute, so a direct dissemination of scientific publications is possible.

Since these plugins are implemented into a website as an iFrame, it is not possible to connect the loaded data to the automated notifications due to Cross-Origin Resource Sharing.
